| 기능 | 설명 |
|------|------|
| 📝 메모 | 날짜별 카드형 메모 — 여러 개 추가, 개별 수정·삭제 |

```
│       달력          │  📝 메모   │              │

```

## 📖 사용법

### 메모
1. 달력에서 날짜 클릭
2. 오른쪽 메모 탭 하단 입력창에 내용 작성
3. **추가** 버튼 클릭 → 카드로 추가됨
4. 카드의 **✏** 로 수정, **✕** 로 삭제

### D-day
1. D-day 탭에서 이름과 날짜(YYYY-MM-DD) 입력 후 **추가**
2. 달력에 D-day 이름이 표시됨
3. 3일 이내로 다가오면 앱 시작 시 알림

### 리마인더
1. 알림 탭에서 내용과 시각(HH:MM) 입력 후 **추가**
2. 매일 해당 시각에 팝업 알림 표시
3. 토글로 개별 켜기/끄기 가능
> ⚠ 앱이 실행 중일 때만 작동합니다.

### TODO
1. TODO 탭에서 날짜 선택 후 할 일 입력 → **Enter**
2. 체크박스 클릭으로 완료 처리
3. 통계 탭에서 완료율 확인

## ⚙ 설정

| 설정 | 위치 |
|------|------|
| 🎨 테마 변경 | 메뉴 → 설정 → 🎨 테마 |
| 🔤 폰트 변경 | 메뉴 → 설정 → 🔤 폰트 설정 |
| 🔆 투명도 조절 | 메뉴 → 설정 → 🔆 투명도 (50~100%) |
| 📌 항상 위에 고정 | 메뉴 → 설정 → 📌 항상 위에 고정 |
| 🖥 바탕화면에 고정 | 메뉴 → 설정 → 🖥 바탕화면에 고정 |

### 바탕화면 고정 모드
- 창 프레임 없이 바탕화면 위젯처럼 표시
- 가장자리 드래그로 크기 조절, 내부 드래그로 이동
- **우클릭** → 설정 열기 / 고정 해제

---

## 💾 데이터 관리

- **내보내기** — 메뉴 → 설정 → 📤 데이터 내보내기 → JSON 파일로 백업
- **가져오기** — 메뉴 → 설정 → 📥 데이터 가져오기 → 기존 데이터 교체
- 데이터 저장 위치: 실행 파일과 같은 폴더의 `diary.db`
