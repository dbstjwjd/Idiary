앱 개요

📝

메모

날짜별로 메모를 카드 형식으로 여러 개 등록하고 개별 수정·삭제할 수 있습니다.

📅

D-day

기념일·시험·여행 등 중요한 날까지 남은 날 수를 실시간으로 확인합니다.

⏰

리마인더

매일 지정한 시각에 알림이 울려 중요한 일을 놓치지 않게 합니다.

✅

TODO

주 단위로 할 일을 관리하고 완료 체크할 수 있습니다.

📊

통계

월별 할일 완료율을 그래프로 한눈에 확인할 수 있습니다.

🎨

7가지 테마

IDIS, Dark, Light, Sky, Blossom, Mint, 고양이 테마 중 선택할 수 있습니다.

빠른 시작

앱 실행

Idiary.exe 를 더블클릭하거나 python main.py로 실행합니다.

날짜 클릭

달력에서 원하는 날짜를 클릭하면 오른쪽 메모 탭에 해당 날짜의 메모가 표시됩니다.

메모 추가

메모 탭 하단 입력창에 내용을 입력하고 추가 버튼을 누릅니다.

사이드바로 기능 전환

오른쪽 사이드바의 아이콘 버튼으로 메모 / D-day / 알림 / 통계 / TODO 탭을 전환합니다.

기능 설명

📅 달력

  • 단일 클릭 — 날짜를 선택하고 해당 날짜의 메모를 오른쪽 패널에 표시합니다.
  • 더블 클릭 — 동일하게 날짜를 선택합니다.
  • 메모가 있는 날짜에는 점(•) 이 표시됩니다.
  • D-day로 등록한 날짜에는 이름이 달력 안에 표시됩니다.
  • ◀ ▶ 버튼으로 월을 이동합니다.
  • 달력과 오른쪽 패널 사이 구분선을 드래그해서 가로 비율을 조절할 수 있습니다.

📝 메모

  • 날짜별로 여러 개의 메모를 카드 형식으로 추가할 수 있습니다.
  • 카드 오른쪽 하단의 ✏ 버튼으로 내용을 수정합니다.
  • ✕ 버튼으로 해당 메모 카드를 삭제합니다.
  • 하단 입력창에 내용을 입력 후 추가 버튼을 누르면 새 카드가 생성됩니다.

📅 D-day

  • 이름과 날짜(YYYY-MM-DD)를 입력하고 추가를 누릅니다.
  • 목록에서 남은 날 수 / D-Day / 지난 날 수가 자동 계산되어 표시됩니다.
  • D-day가 3일 이내로 다가오면 앱 시작 시 알림 팝업이 표시됩니다.
  • 삭제 버튼(✕)으로 개별 삭제합니다.

⏰ 리마인더

  • 알림 내용과 시각(HH:MM)을 입력하고 추가를 누릅니다.
  • 지정한 시각이 되면 화면 우측 하단에 토스트 알림이 표시됩니다.
  • 토글 버튼으로 알림을 켜거나 끌 수 있습니다.
  • 같은 알림은 하루에 한 번만 울립니다. (앱 실행 중에만 작동)

✅ TODO

  • 오른쪽 사이드바의 ✅ TODO 탭에서 할 일을 관리합니다.
  • 상단 요일 버튼으로 날짜를 선택하고, 입력창에 할 일을 입력 후 Enter로 추가합니다.
  • 체크박스를 클릭하면 완료 처리되며, 통계 탭의 완료율이 즉시 반영됩니다.
  • ◁ ▷ 버튼으로 주를 이동합니다.
  • 미완료 할 일이 있으면 앱 시작 시 알림이 표시됩니다.

📊 통계

  • 월별 할 일 전체 개수와 완료 개수, 완료율을 보여줍니다.
  • TODO를 완료/삭제하면 즉시 통계에 반영됩니다.

설정

🎨 테마

🏢 IDIS
🌙 Dark
☀️ Light
☁ Sky
🌸 Blossom
🌿 Mint
🐱 고양이

상단 메뉴 → 설정 → 🎨 테마에서 변경합니다. 선택한 테마는 자동 저장됩니다.

🔤 폰트 설정

  • 설정 → 🔤 폰트 설정에서 서체와 크기를 변경합니다.
  • 앱 전체 텍스트에 즉시 적용됩니다.

🔆 투명도

  • 설정 → 🔆 투명도에서 50% ~ 100% 사이로 창 투명도를 조절합니다.

📌 항상 위에 고정

  • 다른 창 위에 항상 표시합니다. 설정 메뉴에서 체크로 켜고 끕니다.
  • 바탕화면 고정 모드와 동시에 사용할 수 없습니다.

🖥️ 바탕화면에 고정

  • 창 프레임이 없어지고 바탕화면에 붙어있는 위젯처럼 표시됩니다.
  • 창 가장자리를 드래그해서 크기를 조절하고, 내부를 드래그해서 위치를 이동합니다.
  • 우클릭 메뉴로 설정을 열거나 고정을 해제할 수 있습니다.
TIP — 바탕화면 고정 모드에서는 상단 메뉴바가 숨겨집니다. 우클릭 → 설정 열기로 접근하세요.

데이터 관리

📤 데이터 내보내기

  • 설정 → 📤 데이터 내보내기를 클릭합니다.
  • 저장 위치를 선택하면 JSON 파일로 백업됩니다.
  • 메모, 할일, D-day, 리마인더 데이터가 모두 포함됩니다.

📥 데이터 가져오기

  • 설정 → 📥 데이터 가져오기를 클릭합니다.
  • 기존에 내보낸 JSON 파일을 선택합니다.
  • 주의 — 기존 데이터가 모두 덮어씌워집니다. 가져오기 전에 먼저 내보내기로 백업하세요.
데이터 저장 위치 — 실행 파일(.exe)과 같은 폴더의 diary.db 파일에 저장됩니다.
